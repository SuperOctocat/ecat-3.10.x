About the Data and Publications Search tool

The role of Data and Publications Search tool is to hold the main catalogue of Geoscience Australia's datasets, services and some other classes of items. It delivers a subset of Geoscience Australia's main catalogue that is available to the general public. Data and Publications Search tool Public operates within a collection of catalogues:


eCat Public - a subset of published metadata records visible on the web. This filters out business metadata not required for consumption externally. External metadata records have completed a release workflow which includes approval for publication

eCat Non-Public - all of Geoscience Australia's metadata records, both published and not, visible within Geoscience Australia Source Data Catalogue - For data coming in to, or raw data being generated by, Geoscience Australia

Technical information

Geoscience Australia's Data and Publications Search tool is a customised open source application called "GeoNetwork". It has been modified to to incorporate Geoscience Australia's business, technical and user requirements.
